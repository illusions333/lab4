Задачі для користувачів, щоб перевірити вивчений матеріал:

У трикутнику ABC, де кут C є прямим, довжина сторони AC дорівнює 6 см, а довжина катету BC дорівнює 8 см. Знайдіть довжину сторони AB. Проведіть висоту СH з вершини С на сторону АB. Обчисліть довжину цієї висоти.











Розв'язок неправильний!
Правильний розв'язок:



1) За теоремою Піфагора: АВ² = AC² + BC² = 36 + 64 = 100 => AB = 10 (см)

2) S(ABC) = (1/2) * AC * BC = (1/2) * AB * CH => CH = AC * BC / AB = 6 * 8 / 10 = 4.8 (см) АБО

ВН = х; AH = 10 - x

CHB: CH² = 36 - x²

AHC: CH² = 64 - (10 - x)²

36 - x² = 64 - (10 - x)² => 36 - x² = 64 - 100 + 20x - x² => 20x = 72 => x = 3.6 => CH = sqrt(36 - 12.96) = sqrt(23.04) = 4.8 (см)
